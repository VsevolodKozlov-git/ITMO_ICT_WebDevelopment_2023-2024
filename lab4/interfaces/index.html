    Раздел reader

    Все, что связано с созданием и получением информации о читателе

    /reader/book_month_ago

    Описание

    Возвращает список пользователей, которые взяли книгу больше месяца назад, и их контактные данные

    Используемые вызовы api

    /lab3/reader/book_month_ago

    /reader/rare_books

    Описание:

    Выводит список читателей с книгами, которых не более 2-х экземпляров

    Используемые вызовы api:

    lab3/reader/rare_books

    /reader/outdated

    Описание: 

    Страница со списком пользователей, которые не прошли перерегистрацию

    Используемые вызовы api:

    lab3/reader/outdated

    /reader/create

    Описание: 

    Добавление нового читателя в систему

    Используемые вызовы api:

    lab3/reader/create

lab3/book_instance/{id}
